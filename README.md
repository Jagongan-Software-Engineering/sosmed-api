# Sosmed API

Repository Ngoprek Backend ExpressJS

Cara Menjalankan

1.  Rename file `.env.example` menjadi `.env`
2.  Isi field pada file `.env` sesuai dengan data kalian.
3.  Install Dependency
    `npm install`
4.  Start Production Server
    `npm start`

5.  Start Development Server
    `npm run dev`
